### *Installation process* :
- clone repo
- navigate to project root folder
- install all dependencies with yarn or npm i
- run with yarn start or npm


```bash
$ git clone https://github.com/RandolphG/hangman.git
$ cd project folder
$ yarn or npm i
$ yarn start or npm start
```
